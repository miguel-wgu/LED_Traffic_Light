# RGB Traffic Light

An Arduino project that simulates a traffic light using 3 LEDs.

## Demo

![LED Cycle](docs/LED%20Cycle.GIF)

## Hardware

- Elegoo Megar R3 (Arduino Mega 2560)
- 3x LEDs (Green, Yellow, and Red)
- 3x 220ohm Resistors
- Jumper wires
- Breadboard
