# RGB Traffic Light

An Arduino project that simulates an RGB traffic light using 3 RGB LEDs.

## Hardware

- Elegoo Megar R3 (Arduino Mega 2560)
- 3x RGB LEDs (Green, Yellow, and Red)
- 3x 220ohm Resistors
- Jumper wires
- Breadboard
